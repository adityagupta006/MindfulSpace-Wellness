# MindfulSpace – Mindfulness & Wellness App

MindfulSpace is a mental wellness application designed to help users 
build healthy mental habits through guided mindfulness practices. 
Built as a B.Tech 2nd Year project, the app focuses on providing 
a calm and intuitive experience for users seeking emotional balance 
and mental clarity in their daily lives.

## Features
- 🧘 Guided Meditation – Step-by-step meditation sessions for beginners
- 😊 Mood Tracker – Daily mood logging to track emotional patterns
- 🌬️ Breathing Exercises – Calming breath techniques to reduce stress
- 💬 Daily Affirmations – Positive messages to start the day right

## Design Highlights
- Soothing color palette for a calm user experience
- Smooth animations and intuitive navigation
- Mobile-first UI/UX design principles
